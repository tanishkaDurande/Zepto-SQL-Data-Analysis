
# 🛒 Zepto Inventory & Sales Data Analysis using PostgreSQL

## 📌 Project Overview

This project analyzes a Zepto inventory dataset using PostgreSQL. It focuses on data exploration, data cleaning, and solving business-related problems through SQL queries. The project was completed as part of my SQL learning journey to strengthen my database and data analysis skills.

## 🎯 Objectives

* Create and manage a relational database
* Explore and clean the dataset
* Analyze inventory and pricing data
* Generate business insights using SQL

## 🛠️ Tools & Technologies

* PostgreSQL
* SQL
* pgAdmin 4

## 📊 Key Analysis Performed

* Data exploration and validation
* Data cleaning and transformation
* Top discounted products
* Estimated revenue by category
* High-MRP out-of-stock products
* Average discount by category
* Price-per-gram analysis
* Inventory weight analysis
* Additional analytical SQL queries

## 💡 SQL Concepts Used

* CREATE TABLE
* SELECT, WHERE, ORDER BY
* GROUP BY, HAVING
* DISTINCT
* CASE Statements
* COUNT(), SUM(), AVG()
* UPDATE & DELETE

## 📚 What I Learned

* Database creation and management
* Data cleaning using SQL
* Writing analytical SQL queries
* Generating business insights from data
* Improving problem-solving skills using SQL
